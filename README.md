# Hotelito 🏨

Proyecto desarrollado para la asignatura de Programación II. Se trata de una aplicación de escritorio construida en C# con Windows Forms y la biblioteca Bunifu, que simula la gestión de reservas en un hotel ya que la base de datos se usa mediante archivos de txt. Esto por los objetivos de la asignatura.

## 🛠️ Tecnologías Usadas

- **Lenguaje:** C#
- **Interfaz:** Windows Forms con Bunifu
- **Base de datos:** Archivos locales en la carpeta `Base de datos hotel`

## ⚙️ Instalación

1. Clona el repositorio:  
   ```bash
   git clone https://github.com/CyberMedina/Hotelito.git
   ```
2. Mueve la carpeta `Base de datos hotel` al **Disco Local C** (`C:\Base de datos hotel`).
3. Compila y ejecuta el proyecto en Visual Studio.
